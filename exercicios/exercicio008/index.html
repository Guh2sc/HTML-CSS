Principais formatodos de texto:

Negrito/Destaque

Nessa frase, temos um termo em negrito usando a tag <b>(não semântica).

Nesse frase, temos termo em destaque usando a tag <STRONG>.(sêmantica)

Itálico/Ênfase

Nessa frase, temos um termo em italico, usando a tag <I>. ( não semantico)

Nessa frase, temos um termo em ênfase usando a tag <EM>. (sêmantica)

Texto marcado

Podemos criar também um texto marcado usando a tag <MARK> (mudar a cor com CSS)

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno (<SMALL>)nesse parágrafo.

Texto deletado

Podemos marcar um texto excluído para indicar que ele deve ser lido, mas não considerando. Usano a tag <DEL>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.<INS>

Existe tambem o sublinhado utlizando a tag <U>( não semantico).

Texto sobrescrito

Para inserir coisas do tipo x20+3. (<SUP>)

Texto subscrito

para inserir coisas do tipo H2O.(<SUB>)
